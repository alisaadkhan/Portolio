> **"Code is the easy part. Architecture is the flex."**

This repository contains the source code for my personal portfolio website. Designed with a cyber-security inspired aesthetic, it prioritizes performance, accessibility, and a distinct professional identity suited for a Full Stack Solutions Architect.

## âš¡ Key Features

* **Tactical Grid System:** A pure CSS background pattern that mimics blueprint architecture without image assets.
* **High-Performance Animations:** All movements (Marquee, Hovers) use `transform: translate3d` for GPU acceleration and 60FPS smoothness.
* **Terminal Interactions:** Custom-styled contact inputs and "Identity Block" that reinforce the Command Line Interface (CLI) vibe.
* **Responsive Design:** A split-screen layout (Bento Grid style) that adapts seamlessly from 4K monitors to mobile devices.
* **Modern Stack:** Built with React, TypeScript, and Tailwind CSS for rapid, reliable delivery.
